# Папка для ассетов мода

Сюда нужно скидывать все текстуры перед тем, как попросить агента добавить их в мод.

## Структура папок

```
assets-input/
├── items/          ← текстуры предметов (посохи, кольца, и т.д.)
├── blocks/         ← текстуры блоков (если будут)
└── armor/          ← текстуры брони/мантий
```

## Правила именования

Имя файла должно совпадать с ID предмета в коде.

| Предмет              | Папка           | Имя файла           |
|----------------------|-----------------|---------------------|
| Посох огня           | `items/`        | `fire_staff.png`    |
| Посох воды           | `items/`        | `water_staff.png`   |
| Огненная мантия      | `armor/`        | `fire_robe.png`     |

## Требования к текстурам

- Формат: **PNG**
- Рекомендуемый размер: **16×16** или **32×32** пикселя

## Как использовать

1. Закинь `.png` в нужную папку (например, `items/fire_staff.png`)
2. Напиши агенту: «Перенеси текстуры из assets-input» или «Добавь посох огня с моей текстурой»
3. Агент автоматически найдёт файл и поместит его в правильное место
