# Папка для текстур предметов

Сюда нужно класть `.png` файлы текстур предметов мода.

## Правила именования

Имя файла должно совпадать с именем предмета:

| Предмет         | Имя файла          |
|-----------------|--------------------|
| Посох огня      | `fire_staff.png`   |

## Требования к текстурам

- Формат: **PNG**
- Рекомендуемый размер: **16×16** или **32×32** пикселя
- Размер должен быть кратен 16

## Как это работает

Модель предмета (`models/item/fire_staff.json`) автоматически подтягивает текстуру с именем `fire_staff.png` из этой папки. Просто добавь файл — и игра подхватит его при следующем запуске.
